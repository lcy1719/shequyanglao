## 【功能介绍】
**系统包含老人、员工、管理员三个角色
老人：登录、老人主页、活动列表、活动申请、员工评价、健康档案、我的活动、个人信息等功能
员工：登录、员工主页、新增档案、新增活动、审批报名等功能
